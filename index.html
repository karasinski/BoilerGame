Welcome to the BoilerGame repo.

BoilerGame is my attempt at creating a complete game with the Phaser framework. I've included a main menu, instructions page, and leaderboard (with localhost saving ability). The example game is a modified version of the tutorial by Loopeex.

A complete walkthrough of the code can be found here.

Don't want to read? Go ahead and play the game.
